# STACK_E_ESTRUTURA.md

Decisao tecnica para o projeto sid.dev.br.

## Decisao principal

O site sid.dev.br pode usar como inspiracao a estrutura e as ferramentas de sites modernos como buttermax.net, mas nao deve copiar cores, imagens, textos, codigo, layout exato ou identidade visual.

## O que pode ser usado como inspiracao

Podemos usar a ideia de:

- Site fluido
- Animacoes suaves
- Interacao com mouse
- Rotas internas
- Estrutura em componentes
- Seccoes animadas
- Cards de projetos
- Fundos interativos
- Animações vetoriais estilo Lottie
- Transicoes modernas
- Sensacao de portfolio premium

## O que deve ser diferente

O sid.dev.br deve ter identidade propria:

- Cores proprias
- Imagens proprias
- Textos proprios
- Projetos do Sid
- Livros do Sid
- Musicas do Sid
- Ecossistema SID
- Visual autoral
- Tom tecnologico, humano e criativo

## Stack recomendada para versao profissional

A versao inicial pode continuar em HTML, CSS e JavaScript puro.

A versao profissional futura pode usar:

- Next.js
- React
- TypeScript
- Tailwind CSS
- Motion ou Framer Motion
- Lottie ou dotLottie
- Markdown ou MDX para conteudo
- Vercel ou GitHub Pages para publicacao
