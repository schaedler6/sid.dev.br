Referencias visuais do projeto sid.dev.br.

## Decisao sobre Buttermax

O Sid gostou da estrutura, fluidez, movimento e sensacao premium do buttermax.net.

Podemos usar como referencia:

- Estrutura moderna
- Interacao com mouse
- Animações suaves
- Rotas internas
- Componentizacao
- Experiencia fluida
- Possivel uso de Lottie
- Sensacao de laboratorio digital criativo

Nao usar:

- Cores originais
- Imagens originais
- Textos originais
- Codigo copiado
- Layout identico
- Identidade visual copiada

Direcao correta:

Criar uma versao propria para o universo SID, com cores, imagens, conteudo e identidade autorais.
